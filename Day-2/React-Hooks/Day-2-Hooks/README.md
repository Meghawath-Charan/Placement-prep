# Day 2 — React Hooks Deep Dive

Advanced hook patterns with real-world applications.

## Components

### ✅ Counter Component
- **File:** `src/components/Counter.jsx`
- **Concepts:** `useState` with functional updates, state history
- **Features:** Increment/Decrement buttons, history tracking

### ✅ Users Component
- **File:** `src/components/Users.jsx`
- **Concepts:** `useEffect`, API fetching, dependency arrays, cleanup functions
- **Features:** Async data loading, error handling, loading states

### ✅ LiveSearch Component
- **File:** `src/components/LiveSearch.jsx`
- **Concepts:** Controlled components, real-time filtering
- **Features:** Case-insensitive search, dynamic list rendering

## Run Project

```bash
npm install
npm run dev
```
